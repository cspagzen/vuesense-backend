## 1. BOARD NAVIGATION & INTERACTION\
\
### 1.1 UNDERSTANDING THE BOARD LAYOUT\
\
**The Portfolio Board Structure:**\
- **Rows 1-5 (NOW/NEXT)**: Above the Mendoza Line - work you CAN deliver\
- **Row 6+ (LATER)**: Below the Mendoza Line - work you CANNOT deliver yet\
- **Columns A-H**: Priority within each row (rightmost = highest priority in that row)\
- **Initiative Cards**: Blue (Strategic), Yellow (KTLO), Purple (Emergent)\
- **Team Health Cards**: Show which teams work on which initiatives\
\
**Visual Indicators:**\
- **Validation Dots**: Red (not validated), Teal (in validation), Green (validated)\
- **Team Card Colors**: White text = actively working, Faded text = slated for future\
\
### 1.2 DRAG AND DROP INITIATIVES\
\
**How to Move an Initiative:**\
1. **Click and hold** on any initiative card\
2. **Drag** to the desired position (ghost preview shows placement)\
3. **Release** to drop in new position\
4. Cards automatically reflow to maintain row constraints\
\
**Drag and Drop Rules:**\
- Row 1: Maximum 1 initiative\
- Row 2: Maximum 2 initiatives  \
- Row 3: Maximum 3 initiatives\
- And so on...\
- Within a row: Rightmost position = highest priority\
\
**Keyboard Alternative:**\
- Tab to initiative, press Space to select\
- Use arrow keys to move position\
- Press Enter to confirm placement\
\
**What Happens When You Move:**\
- Team assignments follow the initiative\
- Risk scores recalculate automatically\
- Mendoza Line implications update\
- Dashboard metrics refresh in real-time\

## 2. VIEWING & EDITING INITIATIVE DETAILS\
\
### 2.1 VIEWING INITIATIVE INFORMATION\
\
**Quick View - Hover:**\
1. **Hover** over any initiative card\
2. See tooltip with: Title, Type, Validation Status, Teams\
3. Related team cards highlight automatically\
\
**Detailed View - Click:**\
1. **Click** on any initiative card\
2. Modal opens with full details:\
   - Initiative Canvas information\
   - Assigned teams with health status\
   - JIRA integration data\
   - Risk score breakdown\
   - Progress metrics\
   - Comments and blockers\
\
### 2.2 EDITING INITIATIVE INFORMATION\
\
**From the Detail Modal:**\
1. Click on an initiative to open modal\
2. Click **"Edit Initiative"** button (pencil icon)\
3. Modify fields:\
   - Title and description\
   - Type (Strategic/KTLO/Emergent)\
   - Validation status\
   - Team assignments\
   - Progress percentage\
   - Comments\
4. Click **"Save Changes"** to apply\
5. Click **"Cancel"** to discard changes\
\
**Quick Edit Priority:**\
1. Simply drag the initiative to a new position\
2. No need to open the modal for priority changes\
\

## 3. TEAM HEALTH MANAGEMENT\
\
### 3.1 VIEWING TEAM HEALTH\
\
**Quick View on Board:**\
- Look at team health cards next to initiatives\
- 3 indicators visible: Capacity, Skillset, Leadership/Vision\
- Red = Critical, Orange = At Risk, Green = Healthy\
\
**Detailed Team View:**\
1. **Click** on any team health card\
2. Modal shows all 6 health dimensions:\
   - **Capacity**: Team bandwidth\
   - **Skillset**: Technical capabilities  \
   - **Vision**: Clarity of direction\
   - **Support**: Tools and resources\
   - **Team Cohesion**: Collaboration quality\
   - **Autonomy**: Decision-making independence\
3. View utilization percentage\
4. Read team comments and notes\
\
### 3.2 UPDATING TEAM HEALTH\
\
**Edit Team Health Dimensions:**\
1. Click on a team card to open modal\
2. Click **"Edit Health"** button\
3. For each dimension, select status:\
   - Healthy (Green)\
   - At Risk (Orange)\
   - Critical (Red)  \
   - Not Set (Gray)\
4. Update utilization percentage (0-150%)\
5. Add/edit team comments\
6. Click **"Save Changes"**\
\
**Keyboard Shortcuts in Edit Mode:**\
- Tab: Move between dimensions\
- Arrow keys: Change status\
- Enter: Save changes\
- Escape: Cancel editing\

## 4. OKR TRACKING & KEY RESULTS\
\
### 4.1 VIEWING OKRs\
\
**From the Dashboard (Bento Grid):**\
1. Look for the three KPI gauge cards\
2. Each shows:\
   - Current value (center number)\
   - Target value (outer ring)\
   - Progress percentage (colored arc)\
   - 90-day trend indicator\
\
**Detailed KR View:**\
1. **Click** on any KPI gauge card\
2. Modal opens with:\
   - Historical trend chart\
   - Progress timeline\
   - Pace analysis (current vs required)\
   - Data quality score\
   - Projected completion date\
\
### 4.2 UPDATING KEY RESULTS\
\
**Update Current Value:**\
1. Click on a KPI gauge card\
2. Click **"Update Value"** button\
3. Enter new current value\
4. Optionally add a note about the update\
5. Click **"Save"**\
\
**View History:**\
1. Click on KPI gauge\
2. Click **"View Details"**\
3. See full historical data with:\
   - Date of each update\
   - Value changes over time\
   - Who made updates (if tracked)\

## 6. SEARCH & FILTER FUNCTIONALITY\
\
### 6.1 GLOBAL SEARCH\
\
**Using the Search Bar (Sidebar):**\
1. Click search icon or press **Ctrl/Cmd + K**\
2. Type search query\
3. See instant suggestions:\
   - Initiatives (by title, type, team)\
   - Teams (by name, health status)\
   - Quick filters\
4. Click result or press Enter to apply\
\
**Search Shortcuts:**\
- **"@"** prefix: Search teams only\
- **"#"** prefix: Search initiatives only  \
- **"critical"**: Show all critical items\
- **"blocked"**: Show blocked work\
\
### 6.2 ADVANCED FILTERS\
\
**Opening Filter Panels:**\
1. Click **"Filter"** button in sidebar\
2. Or click filter icon on chip bar\
3. Choose "Portfolio Filters" or "Team Filters"\
\
**Portfolio Filters:**\
- Initiative Type (Strategic/KTLO/Emergent)\
- Validation Status  \
- Priority Range (rows)\
- Progress Range (0-100%)\
- Blocked Stories threshold\
\
**Team Filters:**\
- Overall Health Level\
- Specific Health Indicators\
- Utilization Range\
- Workload (initiative count)\
\
**Applying Filters:**\
1. Check desired filter options\
2. Filters apply immediately\
3. Active filters show as chips below header\
4. Click "X" on chip to remove that filter\
5. Click "Clear All" to reset\
\

## 9. DATA SYNCHRONIZATION\
\
### 9.1 JIRA SYNC\
\
**Automatic Sync:**\
- Occurs every 15 minutes when tab is active\
- Pauses when tab is in background\
- Resumes immediately when you return\
\
**Manual Sync:**\
1. Click **"Sync with Jira"** in sidebar\
2. Or press **Ctrl/Cmd + R**\
3. Sync overlay shows progress\
4. Usually completes in 2-3 seconds\
\
**What Gets Synced:**\
- Initiative progress and status\
- Team velocity and utilization\
- Blocked/flagged stories\
- Epic and story updates\
- Comments from Jira\
\
### 9.2 SYNC INDICATORS\
\
**Sync Status Visual Cues:**\
- **Green pulse**: Recently synced (<5 min)\
- **Yellow pulse**: Needs sync soon (5-15 min)\
- **Red pulse**: Overdue for sync (>15 min)\
- **Spinning icon**: Sync in progress\
\

## 12. COMMON TASKS - STEP BY STEP\
\
### 12.1 "How do I add a new initiative?"\
\
**Method 1 - From Sidebar:**\
1. Click **"Add Initiative"** in sidebar\
2. Fill in the form:\
   - Title (required)\
   - Type (Strategic/KTLO/Emergent)\
   - Description\
   - Initial validation status\
3. Click **"Create"**\
4. New initiative appears in row 1 or lowest priority\
\
**Method 2 - Quick Add:**\
1. Press **Ctrl/Cmd + N**\
2. Fill quick-add form\
3. Press Enter to create\
\
### 12.2 "How do I move an initiative to higher priority?"\
\
1. **Locate** the initiative on the board\
2. **Click and drag** the card upward\
3. **Drop** in a higher row (lower number = higher priority)\
4. Within the row, drag right for highest priority in that row\
\
### 12.3 "How do I mark a team as having capacity issues?"\
\
1. **Click** on the team health card\
2. Click **"Edit Health"** button\
3. Find **"Capacity"** dimension\
4. Select **"Critical"** (red) or **"At Risk"** (orange)\
5. Add a comment explaining the issue\
6. Click **"Save Changes"**\
\
### 12.4 "How do I update our OKR progress?"\
\
1. **Navigate** to Portfolio Pulse view\
2. **Click** on the relevant KPI gauge\
3. Click **"Update Value"**\
4. Enter the new current value\
5. Add an optional note\
6. Click **"Save"**\
\
### 12.5 "How do I see why delivery confidence is low?"\
\
1. Go to **Portfolio Pulse** view\
2. **Click** on the Delivery Confidence card\
3. Modal shows breakdown:\
   - Capacity risks (-X%)\
   - Skill gaps (-X%)\
   - Blocked work (-X%)\
   - Below-line distractions (-X%)\
4. Each factor is explained with specifics\
\
### 12.6 "How do I filter to see only critical teams?"\
\
**Method 1 - Dashboard:**\
1. In Portfolio Pulse, find Team Health card\
2. Click on the "Critical" bar\
3. View opens showing only critical teams\
\
**Method 2 - Filters:**\
1. Click **"Filter"** in sidebar\
2. Select **"Team Filters"**\
3. Under "Overall Health", check "Critical"\
4. Apply filter\
\
### 12.7 "How do I find all blocked initiatives?"\
\
1. Click in search bar or press **Ctrl/Cmd + K**\
2. Type **"blocked"**\
3. Select "Show blocked initiatives" from suggestions\
4. Board filters to show only blocked work\
\
### 12.8 "How do I export or share the board?"\
\
1. Click **"Share"** button in top toolbar\
2. Choose export format:\
   - PDF (for printing)\
   - Image (PNG for presentations)\
   - Link (shareable URL)\
3. Set options (include team health, etc.)\
4. Click **"Export"** or **"Copy Link"**\
\

## 13. TROUBLESHOOTING COMMON ISSUES\
\
### 13.1 "Why can't I drag an initiative?"\
\
**Possible Reasons:**\
- You're in a read-only view\
- The board is locked for sync\
- You don't have edit permissions\
- Browser issue (try refresh)\
\
**Solution:**\
1. Check for sync spinner (wait if syncing)\
2. Verify you're logged in\
3. Try refreshing the page (Ctrl/F5)\
4. Check browser console for errors\
\
### 13.2 "Why don't I see the Mendoza Line?"\
\
The Mendoza Line appears between rows 5 and 6 by default. If missing:\
1. Ensure you have initiatives in row 6 or below\
2. Check if view is zoomed out too far\
3. Try toggling to board view and back\
\
### 13.3 "Changes aren't saving"\
\
1. Check for sync errors (red banner at top)\
2. Verify internet connection\
3. Look for "Demo Mode" indicator\
4. Try manual sync (Ctrl/Cmd + R)\
5. Check browser dev console for errors\
\
### 13.4 "Modal won't close"\
\
1. Press **Escape** key\
2. Click the overlay (dark background)\
3. If frozen, refresh the page\
4. Check for required fields if in edit mode\
\
### 13.5 "Search isn't finding anything"\
\
1. Clear search field and try again\
2. Check active filters (might be limiting results)\
3. Try simpler search terms\
4. Use prefixes (@team or #initiative)\
